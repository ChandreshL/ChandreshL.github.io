Study aid (Ionic)

A cross platform mobile app project during my masters. An App for universities that use Moodle learning management system.

Framework: Ionic 3 Front end: Angular Back end: Tyepscript
Uses REST API to send and receive Json object to Moodle Server.

Github:Source code

An App for universities that uses Moodle learning management system.

Implemented features

  • The app allows students to search, enroll and view courses to logged in users.
  • Local notifications for assignment deadlines and other calendar events that are in the Moodle.
  • Search professors or students and add them in contacts.
  • Send and receive messages from the contacts.

Planned features

  • View lectures and exams schedule information.
  • Local notifications for lectures and exam schedule information.
  • Machine learning based auto reply to studentâ€™s questions.

Object Oriented Design

The Moodle functionalities are divided into two classes one for moodle courses and other for messages they both uses API provider and local database provider classes to send API requests and store data into local database (Indexed DB). Database provider inherits Dexie. It is a wrapper library for Indexed DB.

The Object Oriented Design offers following advantages:

  1. Readable code
  2. Extensible for new features
  3. Flexible to changes.
